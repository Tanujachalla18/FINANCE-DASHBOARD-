# FINANCE-DASHBOARD-
Finance Dashboard is an end-to-end Power BI project that analyzes sales, profit, units sold, COGS, customer segments, countries, and discount trends. It transforms financial data into interactive dashboards with KPIs, dynamic slicers, and visual insights, enabling businesses to monitor performance and make data-driven decisions.

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization
- Business Intelligence (BI)

 📈 Dashboard Features
- 📊 Sales KPI Dashboard
- 💰 Profit Analysis
- 📦 Units Sold Tracking
- 💵 Cost of Goods Sold (COGS)
- 🌍 Country-wise Sales Analysis
- 👥 Sales by Customer Segment
- 🏷️ Discount Band Analysis
- 📅 Monthly Sales Trend
- 🎛️ Dynamic Slicers (Country & Year)
- 🔄 Interactive Dashboard Navigation
  

 📊 Key Business Insights
The dashboard enables users to:
- Monitor overall financial performance.
- Compare sales across multiple countries.
- Analyze customer segment contributions.
- Evaluate the impact of discount bands on sales.
- Track monthly sales trends.
- Measure profitability and operational efficiency.
- Support data-driven business decision-making
